# Client Follow-up

Draft a short, professional follow-up email to a client. The user will provide context: the client name, what's being followed up on (unpaid invoice, awaiting feedback, pending approval, etc.), and how long it's been since the last contact.

Guidelines for the draft:
- Subject line: specific, not generic ("Following up: Invoice INV-20260420-001" beats "Checking in").
- Tone: warm, direct, never accusatory — assume the client is busy, not avoiding.
- Body: 3-5 sentences max. State the ask, give the relevant reference (invoice number, project name, deliverable), and propose a clear next step (e.g., "Could you confirm receipt by Friday?").
- Sign-off: friendly but professional. Leave the sender name as `[Your Name]` for the user to fill in.

If the user mentions this is the second or third follow-up, escalate slightly: be more direct about the impact (e.g., delaying project start, blocking other work) without being hostile.

If key context is missing (client name, what's being followed up, time since last contact), ask for all missing fields in one consolidated question — never one at a time.
